# Kasa Events — אתר נחיתה

אתר נחיתה לעסק Kasa Events (השכרת עמדת צילום 360 לאירועים).
האתר הוא דף יחיד (single-page), עצמאי לחלוטין — כל ה-CSS, ה-JavaScript, הלוגו והגופנים מוטמעים ב-`index.html`. הסרטונים והתמונות נטענים מ-Cloudinary.

## תוכן החבילה

| קובץ | תיאור |
|------|--------|
| `index.html` | האתר המלא (עצמאי, כולל הכל) |
| `vercel.json` | הגדרות פריסה ל-Vercel (כותרות אבטחה, כתובות נקיות) |
| `robots.txt` | מאפשר למנועי חיפוש לאנדקס את האתר |
| `README.md` | הקובץ הזה |

## פריסה ל-Vercel

### דרך 1 — גרירה ושחרור (הכי פשוט)
1. היכנס ל-https://vercel.com והתחבר.
2. לחץ על **Add New… → Project**.
3. גרור את כל התיקייה הזו לאזור ההעלאה (או בחר אותה).
4. לחץ **Deploy**. תוך כמה שניות האתר עולה לאוויר.

### דרך 2 — דרך שורת הפקודה (Vercel CLI)
```bash
npm i -g vercel      # התקנה חד-פעמית
cd kasa-events-site  # להיכנס לתיקייה
vercel               # פריסת תצוגה מקדימה
vercel --prod        # פריסה לפרודקשן
```

### דרך 3 — דרך GitHub
1. העלה את התיקייה למאגר GitHub.
2. ב-Vercel: **Add New → Project → Import** את המאגר.
3. אין צורך בהגדרות build (אתר סטטי). לחץ **Deploy**.

## הערות

- **אין שלב build** — זהו אתר סטטי. Vercel יזהה זאת אוטומטית ויגיש את `index.html`.
- **דומיין מותאם אישית:** אחרי הפריסה, ב-Vercel → Settings → Domains אפשר לחבר דומיין משלך.
- **הסרטונים** מתארחים ב-Cloudinary (לא בחבילה הזו), כך שהאתר נשאר קל ומהיר.
